# 图层

如图可见，框选的位置就是图层（没有图层选项的，可以在窗口-图层里打开）

我们可以看到这里一共有四个图层，每一行的结构都是前面有个小眼睛，中间是个缩略图，后面是这个图层的名字。当我们想要修改图层名字的时候，可以双击图层的名字就可以进行修改了

在图层区域从下往上依次是背景，1,2,3，当我们试着把小眼睛点击关闭的时候可以看到图像中的3不见了

移动工具

移动工具

移动工具就是移动某一图层上的所有元素

点击移动工具

然后选中“3”这个图层，用鼠标点击图像上的“3”就可以看到“3”能被移动，同理想移动“2”也是如此

那我们再打开一张图片

那我们可不可以移动这个瓢虫的位置呢？答案是不能的！

可以看到在左侧下面的图层选项卡里只有一个图层，使用移动工具移动的是某一图层上的所有元素，不光瓢虫，叶子也是图层的一部分，所以移动的时候会一起移动，不会单独移动

属性

  1. 自动选择

    一开始想要移动某一元素，是要先找到图层，然后才可以移动

    自动选择则是在图像上找到你想要移动的元素，当你想移动它的时候，就会自动选中相应图层

  2. 图层/组

    现在看到的是四个图层，并没有组，组就是打包，放在一起的意思

    如果我们将2和3放在一个组里，然后进行移动，会是什么效果？

    可以看到，当我们选择图层的时候，可以分别移动2或3的位置

    而当我们选择组再移动的时候2和3的相对位置就改变了

  3. 显示变换控件

    这个选项勾中之后，可以对某一图层的元素进行大小的改变，调整完按Enter就可以了

    但是这个选项一般不会勾选！如果想要调整大小，一般使用自由变换（Ctrl+T）工具

  4. 对齐工具

    1. 顶对齐 垂直居中对齐 底对齐

      顶对齐是对选中的几个图层进行对齐操作，默认与最高的元素所在的图层进行对齐

      （如何选中多个图层？按住ctrl再点击图层就能够实现多选了）

      同理底对齐，默认与最低的元素所在的图层进行对齐

      垂直居中对齐就是将各个图层之间的元素距离纵向取一个平均值，再进行对齐（自己尝试）

    2. 左对齐 水平居中对齐 右对齐

      和上面的原理是一样的~自己尝试着点一点
