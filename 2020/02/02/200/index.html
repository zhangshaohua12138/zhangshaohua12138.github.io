简介

HTML 超文本标记语言，是由一对一对标签组成的，每一对标签拼到一起就形成了这种语言

标签

<html></html>

形如这样的就是标签，所以可以看出，标签是由头标签和尾标签构成

头标签由：左尖角号-标签名-右尖角号 组成

尾标签由：左尖角号-结束符-标签名-右尖角号 组成

每一对标签有每一对标签的特殊功能

<abc></abc>

这个标签就叫abc标签，当然没有什么实际意义

  • 根标签

      <html></html>

    这个标签叫做根标签，树只能在一个根上开枝散叶，所以一个HTML文件只能有一个根标签，且所有的代码都在根标签内部

  • 结构化标签

    <head></head>
<body></body>

​ 结构化标签在HTML文件中必须要有，当把HTML文件看成一个人

​ head就相当于人的思想-外人是看不到的-标签内容服务于浏览器

​ body就相当于人的身体-外人是能看到的-标签内容服务于浏览者

​ 所以，关于浏览器的配置信息要写在head标签里，而需要用户浏览的内容则写在body标签里

到目前为止，就可以写一个简单的页面了

<html>
    <head></head>
    <body>
        lift is shit！
    </body>
</html>

执行一下，可以看到浏览器上显示了也可以试着加点汉字

```
<html>
    <head></head>
    <body>
        lift is shit！美好的生活！
    </body>
</html>
```

如果有执行结果是乱码的，说明你乱码了

  • 为什么：浏览器不认识写的是什么，是英语还是汉语？所以才会出现这个问题

  • 解决思路：告诉浏览器用的是什么语言

  • 如何解决：由于是告诉浏览器用什么语言，所以应该将要求写在head标签里面，要求就用到了新标签

  • meta标签

    <meta>

    这个标签和别的标签不一样，他是自己单独出现的，但是写完还不好用，还会出现乱码

    所以又有新的知识点(往下看，属性)

  • 标题标签 title

    这个标签是用来做什么的呢？

    在看淘宝的时候的这个位置显示的字，就是title标签的功劳

    <html>
    <head>
        <meta charset="utf-8">
        <title>淘宝！</title>
    </head>
    <body>
        lift is shit！
        美好的生活！
    </body>
</html>

  • 段落标签 p

    <p></p>

    段落标签，能让标签里的内容成段展示

    <html>
    <head>
        <meta charset="utf-8">
        <title>淘宝！</title>
    </head>
    <body>
        <p>这是段落标签</p>
        <p>这是段落标签</p>
        <p>这是段落标签</p>
        <p>这是段落标签</p>
    </body>
</html>

  • 标题标签 h1~h6

    <html>
    <head>
        <meta charset="utf-8">
        <title>淘宝！</title>
    </head>
    <body>
        <h1>这是h1</h1>
        <h2>这是h2</h2>
        <h3>这是h3</h3>
        <h4>这是h4</h4>
        <h5>这是h5</h5>
        <h6>这是h6</h6>
    </body>
</html>

    这是一组标签,在word里面也见过类似的样式，有以下三个作用

    • 独成一段
    • 更改字体大小
    • 加粗字体

  • strong/em

    strong将内容进行加粗，em将内容进行倾斜

    <html>
    <head>
        <meta charset="utf-8">
        <title>淘宝！</title>
    </head>
    <body>
        <strong>加粗</strong>
        <em>倾斜</em>
    </body>
</html>

    如果想让字又加粗又倾斜呢？

    <strong><em>又加粗又倾斜</em></strong>

  • 删除标签 del

    现价xxx，原价xxx

    <html>
    <head>
        <meta charset="utf-8">
        <title>淘宝！</title>
    </head>
    <body>
        现价xxx，<del>原价xxx</del>
    </body>
</html>

  • div/span 标签

    本身没有什么特别之处，但是是作为一个容器进行使用

    可以将页面进行分类，结构化，便于维护

    其次呢，还可以对标签进行捆绑操作

    比如将某几行的字改变颜色，就不用一个标签一个标签的修改，只需要放在一个容器里就行

属性

属性的写法：属性名=’属性值’

属性的位置：写在头标签

charset（编码字符集）：作用就是告诉浏览器用的是什么语言

  • gb2312：包含亚裔字符集，但是，不包括繁体字
  • gbk：上一个的扩展，包含了繁体字
  • unicode：万国码，所有国家的语言都包括了
  • utf-8：上一个的升级版，最通用的

所以，代码就优化为：

<html>
    <head>
        <meta charset="utf-8">
    </head>
    <body>
        lift is shit！
        美好的生活！
    </body>
</html>
